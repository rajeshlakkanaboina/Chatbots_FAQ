# Chatbots_FAQ
A simple FAQ Chatbot built using Python and Tkinter.

Features
GUI-based chatbot interface
Answers predefined FAQ questions
Uses NLP (CountVectorizer and Cosine Similarity) for matching user queries
Supports similar or slightly misspelled questions
Technologies Used
Python
Tkinter
NLTK
Scikit-learn
How to Run
Install required libraries: pip install nltk scikit-learn

Run the application: python chatbot_ui.py
